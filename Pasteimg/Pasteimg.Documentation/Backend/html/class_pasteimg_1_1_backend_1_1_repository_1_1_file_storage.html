Detailed Description

Represents a file storage class that implements IFileStorage interface.

FileStorage routing examples:

GetDirectory:

Root: "root"

SubDirectoryDivision: 3

id : "abcdef"

Output: "/root/abc/"

GetPath:

Root: "root"

SubDirectoryDivision: 5

id : "testtest"

extension : "exe"

fileClass : "main"

Output: "/root/testt/testtest_main.exe"

GetPath

Root: "base"

SubDirectoryDivision: 2

id : "example"

extension : null

fileClass : null

Output: "/base/ex/example"

◆ CheckIsAdmin()

ISession Pasteimg.Backend.Logic.AdminLogic.CheckIsAdmin ( string?  sessionKey)
private

Checks if the session identified by the given session key belongs to an admin user. Throws an UnauthorizedException if the session is invalid or the user is not an admin. Returns the session object if the user is an admin.

Parameters
sessionKeyThe session key identifying the user's session.
Returns
The session object if the user is an admin.
